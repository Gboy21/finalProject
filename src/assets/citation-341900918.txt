Nadeeshani, I.H. (2020). On-Road vehicle Breakdown Assistance. 10.13140/RG.2.2.11833.39521. “HelpMe” is a vehicle breakdown Assistance that can be used to find a mechanic basis on user location. It is an Android Application. Therefore, many android users can access this application. This application will help to reduce wasting user time for found a proper mechanic. 
User can register using their details and send request for appointment. After logging to the application user can search mechanic basis on user location and user can search mechanic basis on specialty of the mechanic. User can update, delete their own vehicle information.
There are frequently Ask Question (FAQ). User can ask question and mechanic can reply for that. User can rate the mechanic if satisfy work done.
